
###  [张量](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%BC%A0%E9%87%8F&zhida_source=entity)初步

前面几节我们一直在用矢量语言——梯度、散度、旋度，[矢量点乘](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%9F%A2%E9%87%8F%E7%82%B9%E4%B9%98&zhida_source=entity)和叉乘。这套语言处理[麦克斯韦方程组](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E9%BA%A6%E5%85%8B%E6%96%AF%E9%9F%A6%E6%96%B9%E7%A8%8B%E7%BB%84&zhida_source=entity)的前三个方程非常高效：$\nabla\cdot\mathbf{E}$、$\nabla\times\mathbf{B}$ 这些表达式用[矢量算符](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%9F%A2%E9%87%8F%E7%AE%97%E7%AC%A6&zhida_source=entity)写出来简洁又直观。

但到第四个方程——电磁场的[动量守恒](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%8A%A8%E9%87%8F%E5%AE%88%E6%81%92&zhida_source=entity)——矢量语言就不够用了。动量是一个[矢量](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=5&q=%E7%9F%A2%E9%87%8F&zhida_source=entity)，[动量流](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%8A%A8%E9%87%8F%E6%B5%81&zhida_source=entity)（单位时间流过单位面积的动量）是一个既有方向（动量方向）又有流向（通过哪个面）的量——它是一个[并矢](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%B9%B6%E7%9F%A2&zhida_source=entity)，或者叫[二阶张量](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E4%BA%8C%E9%98%B6%E5%BC%A0%E9%87%8F&zhida_source=entity)。你不能用一个简单的矢量点乘或叉乘来表示它。[麦克斯韦应力张量](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E9%BA%A6%E5%85%8B%E6%96%AF%E9%9F%A6%E5%BA%94%E5%8A%9B%E5%BC%A0%E9%87%8F&zhida_source=entity) $\overset{\leftrightarrow}{\mathbf{T}}$ 就是这样一个东西：$T_{ij}$ 表示“$j$ 方向的动量沿 $i$ 方向流动的[流密度](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E6%B5%81%E5%AF%86%E5%BA%A6&zhida_source=entity)”。

这一节的任务是引入张量的基本语言——克罗内克 $\delta$ 符号和[列维-奇维塔](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%88%97%E7%BB%B4-%E5%A5%87%E7%BB%B4%E5%A1%94&zhida_source=entity) $\epsilon$ 符号——并展示如何用它们高效地处理矢量恒等式。我们的目标不是全面学习[张量分析](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%BC%A0%E9%87%8F%E5%88%86%E6%9E%90&zhida_source=entity)，而是掌握足够多的代数工具，能够在[电磁学](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%94%B5%E7%A3%81%E5%AD%A6&zhida_source=entity)中自信地处理应力张量、能量-[动量张量](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%8A%A8%E9%87%8F%E5%BC%A0%E9%87%8F&zhida_source=entity)，以及电磁波的[能量密度](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E8%83%BD%E9%87%8F%E5%AF%86%E5%BA%A6&zhida_source=entity)和能流密度。

**一、为什么要引入张量指标记号**

用矢量记号写出来的公式很紧凑，但在需要分量操作时——比如证明一个含叉乘的[恒等式](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=2&q=%E6%81%92%E7%AD%89%E5%BC%8F&zhida_source=entity)，或者计算两个矢量的[并积](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%B9%B6%E7%A7%AF&zhida_source=entity)——矢量记号就显得不够用了。我们想要一种既能显式写出分量、又能紧凑地求和的记号。

**[爱因斯坦求和约定](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%88%B1%E5%9B%A0%E6%96%AF%E5%9D%A6%E6%B1%82%E5%92%8C%E7%BA%A6%E5%AE%9A&zhida_source=entity)**解决了这个矛盾。它的规则很简单：任何乘积中如果同一个指标字母出现恰好两次（一次上标一次下标，或者在[欧几里得空间](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E6%AC%A7%E5%87%A0%E9%87%8C%E5%BE%97%E7%A9%BA%E9%97%B4&zhida_source=entity)中，两次都是下标也可以），就默认对该指标从 $1$ 到 $3$ 求和。

比如，矢量 $\mathbf{A}$ 和 $\mathbf{B}$ 的点乘可以写为：

$$\mathbf{A}\cdot\mathbf{B} = \sum_{i=1}^{3} A_i B_i \quad \rightarrow \quad A_i B_i$$

第二个写法里，指标 $i$ 出现了两次，按照求和约定，自动对所有 $i$ 求和。$\sum$ 符号被省略了，但意思没变。这看起来只是省了点笔墨，但到后面处理多个矢量和复杂恒等式时，这种紧凑性会带来实质性的简化。

矢量 $\mathbf{A}$ 的散度在指标记号下写为 $\partial_i A_i$（$\partial_i \equiv \partial/\partial x_i$）。旋度的第 $i$ 个分量呢？它需要一种能表达“交错”结构的符号。

**二、克罗内克 $\delta$ 符号**

**克罗内克 $\delta$ 符号** $\delta_{ij}$ 定义为：

$$\boxed{\delta_{ij} = \begin{cases} 1, & i = j \\ 0, & i \neq j \end{cases}}$$

它的三个指标全是下标（在欧几里得空间中，上下标没有区别）。$\delta_{ij}$ 就是 $3\times3$ [单位矩阵](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%8D%95%E4%BD%8D%E7%9F%A9%E9%98%B5&zhida_source=entity)的 $(i,j)$ 分量。

$\delta_{ij}$ 在求和中的核心作用是“换指标”。比如：

$$\delta_{ij} A_j = A_i$$

因为当 $j$ 被求和时，只有 $j=i$ 那一项活下来。这看起来是平凡的操作，但在复杂的恒等式化简中，$\delta$ 符号能帮我们消去多余的求和，把表达式大幅度简化。

两个 $\delta$ 符号的乘积也有直观的几何意义。考虑 $\delta_{ij}\delta_{ji}$（对 $i,j$ 求和）。首先对 $j$ 求和：$\delta_{ij}\delta_{ji} = \delta_{ii}$。然后对 $i$ 求和：$\delta_{ii} = \delta_{11} + \delta_{22} + \delta_{33} = 3$。一般地，$\delta_{ii} = 3$ ——这是在[三维空间](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E4%B8%89%E7%BB%B4%E7%A9%BA%E9%97%B4&zhida_source=entity)中单位[矩阵的迹](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%9F%A9%E9%98%B5%E7%9A%84%E8%BF%B9&zhida_source=entity)。

**三、列维-奇维塔 $\epsilon$ 符号**

**列维-奇维塔 $\epsilon$ 符号** $\epsilon_{ijk}$ 又称为完全反[对称张量](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%AF%B9%E7%A7%B0%E5%BC%A0%E9%87%8F&zhida_source=entity)，由以下规则定义：

-   $\epsilon_{123} = 1$
-   交换任意两个指标，符号变号：$\epsilon_{ijk} = -\epsilon_{jik} = -\epsilon_{ikj}$
-   如果有两个指标相同，则 $\epsilon_{ijk} = 0$（因为交换相同指标时，$\epsilon$ 必须同时等于它自己和它的相反数）

因此，三个指标的所有 $3^3 = 27$ 种组合中，只有 $6$ 种非零：$\epsilon_{123} = \epsilon_{231} = \epsilon_{312} = 1$，$\epsilon_{132} = \epsilon_{213} = \epsilon_{321} = -1$，其余 $21$ 种全为零。

$\epsilon_{ijk}$ 是表达“反对称结构”的完美工具。[矢量叉乘](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%9F%A2%E9%87%8F%E5%8F%89%E4%B9%98&zhida_source=entity)可以用它写出来：$\mathbf{C} = \mathbf{A}\times\mathbf{B}$ 的第 $i$ 个分量为：

$$C_i = \epsilon_{ijk} A_j B_k$$

验证一下：$C_1 = \epsilon_{1jk} A_j B_k = \epsilon_{123}A_2 B_3 + \epsilon_{132}A_3 B_2 = A_2 B_3 - A_3 B_2$，确实是叉乘的 $x$ 分量。

旋度也同样可以用 $\epsilon$ 符号写出：$\nabla\times\mathbf{A}$ 的第 $i$ 个分量为 $\epsilon_{ijk} \partial_j A_k$（$\partial_j \equiv \partial/\partial x_j$）。

**四、核心恒等式：$\epsilon_{ijk}\epsilon_{imn}$ 的缩并**

这是整个[张量代数](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%BC%A0%E9%87%8F%E4%BB%A3%E6%95%B0&zhida_source=entity)中最重要的一个恒等式。它把两个 $\epsilon$ 符号的乘积（对公共指标求和）转化为 $\delta$ 符号的组合：

$$\boxed{\epsilon_{ijk} \epsilon_{imn} = \delta_{jm}\delta_{kn} - \delta_{jn}\delta_{km}}$$

这里指标 $i$ 是公共的求和指标，$j,k$ 和 $m,n$ 分别来自两个 $\epsilon$ 符号。

**证明思路**：左边对 $i$ 求和后，结果是一个关于 $j,k,m,n$ 四指标的对象。由于 $\epsilon$ 的完全反对称性，这个对象也必须是关于 $(j,k)$ 反对称和关于 $(m,n)$ 反对称的。在所有可能的四指标构型中，满足这些反对称条件的唯一形式（在三维空间中）就是右边给出的两个 $\delta$ 乘积之差。系数可以通过代入特例（如 $j=m=2, k=n=3$）来定出。

这个恒等式可以从两个对称性的角度来记忆。关于 $(j,k)$：交换 $j$ 和 $k$，左边的 $\epsilon_{ijk} \to \epsilon_{ikj} = -\epsilon_{ijk}$（变号），右边 $\delta_{jm}\delta_{kn} - \delta_{jn}\delta_{km} \to \delta_{km}\delta_{jn} - \delta_{kn}\delta_{jm} = -(\delta_{jm}\delta_{kn} - \delta_{jn}\delta_{km})$（也变号）。关于 $(m,n)$ 同理。两个 $\delta$ 乘积的差值形式是唯一满足这两个反对称条件的组合。

进一步缩并会得到更简单的形式。令 $k=n$ 并对 $k$ 求和：

$$\epsilon_{ijk} \epsilon_{imk} = \delta_{jm}\delta_{kk} - \delta_{jk}\delta_{km}$$

$\delta_{kk}=3$，$\delta_{jk}\delta_{km} = \delta_{jm}$（对 $k$ 求和时，$\delta_{jk}$ 把 $k$ 换成 $j$）。所以：

$$\boxed{\epsilon_{ijk} \epsilon_{imk} = 3\delta_{jm} - \delta_{jm} = 2\delta_{jm}}$$

再缩并一次：令 $j=m$ 并对 $j$ 求和，$\epsilon_{ijk} \epsilon_{ijk} = 2\delta_{jj} = 6$。这个结果是预期之中的：三个非零指标的全排列有 $3! = 6$ 种，每种贡献 $1$，共 $6$。

**五、用张量语言重写矢量恒等式**

有了 $\epsilon_{ijk}\epsilon_{imn}$ 恒等式，以前需要逐分量验证的矢量恒等式现在可以用代数简化一步到位。

**5.1 [矢量三重积](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%9F%A2%E9%87%8F%E4%B8%89%E9%87%8D%E7%A7%AF&zhida_source=entity)：$\mathbf{A}\times(\mathbf{B}\times\mathbf{C})$**

用指标记号写出来：$[\mathbf{A}\times(\mathbf{B}\times\mathbf{C})]_i = \epsilon_{ijk} A_j (\mathbf{B}\times\mathbf{C})_k = \epsilon_{ijk} A_j (\epsilon_{kmn} B_m C_n)$。这里 $\epsilon_{kmn}$ 中的 $k$ 是公共指标。把 $\epsilon_{ijk}$ 和 $\epsilon_{kmn}$ 的指标重新排列，利用 $\epsilon$ 的[循环对称性](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%BE%AA%E7%8E%AF%E5%AF%B9%E7%A7%B0%E6%80%A7&zhida_source=entity) $\epsilon_{ijk} = \epsilon_{kij}$，我们得到 $\epsilon_{kij}\epsilon_{kmn}$。应用恒等式（把 $k$ 作为公共求和指标，相当于原公式中的 $i$）：

$$\epsilon_{kij} \epsilon_{kmn} = \delta_{im}\delta_{jn} - \delta_{in}\delta_{jm}$$

代回去：

$$\begin{aligned} [\mathbf{A}\times(\mathbf{B}\times\mathbf{C})]_i &= (\delta_{im}\delta_{jn} - \delta_{in}\delta_{jm}) A_j B_m C_n \\ &= B_i (A_j C_j) - C_i (A_j B_j) \\ &= B_i (\mathbf{A}\cdot\mathbf{C}) - C_i (\mathbf{A}\cdot\mathbf{B}) \end{aligned}$$

这就一步完成了证明：$\mathbf{A}\times(\mathbf{B}\times\mathbf{C}) = \mathbf{B}(\mathbf{A}\cdot\mathbf{C}) - \mathbf{C}(\mathbf{A}\cdot\mathbf{B})$。用传统方法需要展开四个分量再合并，用张量方法三行搞定。

**5.2 旋度之旋度：$\nabla\times(\nabla\times\mathbf{F})$**

$[\nabla\times(\nabla\times\mathbf{F})]_i = \epsilon_{ijk} \partial_j (\epsilon_{kmn} \partial_m F_n) = \epsilon_{kij}\epsilon_{kmn} \partial_j \partial_m F_n$。同样用恒等式，$\epsilon_{kij}\epsilon_{kmn} = \delta_{im}\delta_{jn} - \delta_{in}\delta_{jm}$。代回：

$$\begin{aligned} [\nabla\times(\nabla\times\mathbf{F})]_i &= (\delta_{im}\delta_{jn} - \delta_{in}\delta_{jm}) \partial_j \partial_m F_n \\ &= \partial_i (\partial_j F_j) - \partial_j \partial_j F_i \\ &= [\nabla(\nabla\cdot\mathbf{F}) - \nabla^2\mathbf{F}]_i \end{aligned}$$

所以 $\nabla\times(\nabla\times\mathbf{F}) = \nabla(\nabla\cdot\mathbf{F}) - \nabla^2\mathbf{F}$。这是电磁波推导中最关键的矢量恒等式，用 $\epsilon$ 符号证明它比逐分量展开简洁得多。

**5.3 两个叉乘的点乘**

在[坡印廷定理](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%9D%A1%E5%8D%B0%E5%BB%B7%E5%AE%9A%E7%90%86&zhida_source=entity)的推导中会出现 $(\nabla\times\mathbf{E})\cdot\mathbf{B} - \mathbf{E}\cdot(\nabla\times\mathbf{B})$ 这一形式。用指标记号重写：$\epsilon_{ijk} (\partial_j E_k) B_i - E_i \epsilon_{ijk} \partial_j B_k = \epsilon_{ijk} B_i \partial_j E_k - \epsilon_{ijk} E_i \partial_j B_k$。交换第一项中 $\epsilon$ 的指标 $\epsilon_{ijk} = -\epsilon_{jik}$，它可以写为 $-\partial_j(\epsilon_{jik} E_k B_i) + \epsilon_{jik} E_k \partial_j B_i$。最终这一组合等于 $-\nabla\cdot(\mathbf{E}\times\mathbf{B})$，这正是坡印廷[矢量散度](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E7%9F%A2%E9%87%8F%E6%95%A3%E5%BA%A6&zhida_source=entity)的负值。

**六、并矢与二阶张量**

两个矢量 $\mathbf{A}$ 和 $\mathbf{B}$ 的**并积**（或[张量积](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%BC%A0%E9%87%8F%E7%A7%AF&zhida_source=entity)）记作 $\mathbf{A}\mathbf{B}$，它是一个二阶张量，分量就是 $A_i B_j$。注意这里没有点乘也没有叉乘——就是简单地把两个矢量的分量拼成矩阵。$A_i B_j$ 和 $B_j A_i$ 是不同的分量（一个是矩阵的 $(i,j)$ 元，一个是 $(j,i)$ 元）。

[单位张量](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%8D%95%E4%BD%8D%E5%BC%A0%E9%87%8F&zhida_source=entity)（二阶）就是 $\delta_{ij}$。任意二阶张量 $\overset{\leftrightarrow}{\mathbf{T}}$ 可以分解为对称部分和反对称部分：$T_{ij} = \frac{1}{2}(T_{ij} + T_{ji}) + \frac{1}{2}(T_{ij} - T_{ji})$。

两个矢量的点乘有两种推广方式。矢量 $\mathbf{A}$ 点乘张量 $\overset{\leftrightarrow}{\mathbf{T}}$，得到一个矢量：$(\mathbf{A}\cdot\overset{\leftrightarrow}{\mathbf{T}})_j = A_i T_{ij}$（对第一个指标求和）。反过来，$(\overset{\leftrightarrow}{\mathbf{T}}\cdot\mathbf{A})_i = T_{ij} A_j$（对第二个指标求和）。这两个结果一般不同，除非 $\overset{\leftrightarrow}{\mathbf{T}}$ 是对称的。

张量的散度也是类似的：$(\nabla\cdot\overset{\leftrightarrow}{\mathbf{T}})_j = \partial_i T_{ij}$。这是一个矢量——对张量的第一个指标取散度，得到的是每个 $j$ 分量上的散度值。

**七、麦克斯韦应力张量的指标形式**

现在我们用张量语言来写电磁场的动量守恒。麦克斯韦应力张量是：

$$T_{ij} = \epsilon_0\left( E_i E_j - \frac{1}{2}\delta_{ij} E^2 \right) + \frac{1}{\mu_0}\left( B_i B_j - \frac{1}{2}\delta_{ij} B^2 \right)$$

[动量守恒定律](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E5%8A%A8%E9%87%8F%E5%AE%88%E6%81%92%E5%AE%9A%E5%BE%8B&zhida_source=entity)的微分形式为：

$$\frac{\partial}{\partial t}(\mathbf{g})_i + \partial_j T_{ij} = -(\rho\mathbf{E} + \mathbf{J}\times\mathbf{B})_i$$

左边第一项是动量密度的时间变化率（$\mathbf{g} = \mu_0\epsilon_0 \mathbf{S} = \epsilon_0 \mathbf{E}\times\mathbf{B}$），第二项是动量流密度张量的散度（单位体积内通过表面流入的净动量），右边是[洛伦兹力密度](https://zhida.zhihu.com/search?content_id=279202037&content_type=Article&match_order=1&q=%E6%B4%9B%E4%BC%A6%E5%85%B9%E5%8A%9B%E5%AF%86%E5%BA%A6&zhida_source=entity)的负值（电磁场对物质施加的力，等于场自身动量的减少）。

这个方程的指标形式非常紧凑。推导它需要用到 $\partial_j T_{ij}$ 的展开——把 $T_{ij}$ 代入，计算 $\partial_j T_{ij}$，利用麦克斯韦方程组把散度和旋度项重新组合。整个推导约需半页纸的指标运算，但用纯矢量记号来做则相当冗长。张量指标在这里展现出不可替代的优势。

张量记号引入了一套新的代数语言。它的核心无非是两条：$\delta_{ij}$ 用来“换指标”和表达各向同性的单位结构，$\epsilon_{ijk}$ 用来表达反对称性和叉乘。把这两个符号结合起来，加上 $\epsilon_{ijk}\epsilon_{imn}$ 缩并恒等式，矢量分析中几乎所有恒等式——从三重积到旋度之旋度——都可以用这套代数机械地推出来。在后续章节中，当我们处理电磁场的能量-动量张量和麦克斯韦应力张量时，这套语言将使我们能够从麦克斯韦方程组直接走到守恒定律，而不会在矢量恒等式的丛林里迷失方向。
